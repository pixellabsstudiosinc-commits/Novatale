## 🎯 How to Take Screenshots

### On macOS:
```bash
# Full screen
Cmd + Shift + 3

# Selected area
Cmd + Shift + 4

# Specific window
Cmd + Shift + 4, then press Space, click window
```

### Tips for Great Screenshots:
1. ✅ **Clean your interface** - Close unnecessary windows
2. ✅ **Use sample content** - Create a demo project with real data
3. ✅ **Show key features** - Highlight important functionality
4. ✅ **Consistent style** - Use same window size and theme
5. ✅ **High resolution** - At least 1200px wide

## ⚠️ Fallback Behavior

If screenshots are missing, the landing page will display:
- **"Screenshot Coming Soon"** placeholder
- **Gradient background** matching the brand colors
- **No broken images** - graceful degradation
